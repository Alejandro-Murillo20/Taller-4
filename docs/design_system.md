# Sistema de Diseño


## Paleta de Colores


- **Primario (Acción):** #08026eff (Azul Marino Oscuro) - Usar en botones principales.

- **Secundario (Soporte):** #ce0e0eff (Rojo Intenso) - Usar en mensajes de éxito o confirmación.

- **Fondo General:** #F3F4F6 (Gris Claro) - Para evitar el cansancio visual. 

- **Texto Principal:** #000000 (Negro) - Para legibilidad.

- **Acentos de Error:** #b19f39ff (Amarillo Oliva).


## Tipografía

- **Fuente Principal:** Times New Roman - Para mostrar sobriedad. 

- **Encabezados (H1, H2):** Bold, color oscuro.

- **Cuerpo:** Regular, tamaño 16px para legibilidad.


## Componentes UI

- **Botones:** Bordes redondeados (border-radius: 10px), padding amplio.
