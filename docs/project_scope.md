# Alcance del Proyecto: "Mi Parqueadero"


## Descripción General

Mi Parqueadero es una aplicación diseñada para empresas dedicadas a la administración de parqueaderos que desean tener un sistema más sencillo pero que les brinde seguridad y eficiencia. 


## Usuario Objetivo

Empresas que deseen administrar su negocio de una forma sencilla pero segura. 


## Funcionalidades Principales (Core Features)

1. **Registro de Vehículos:** El usuario podrá registrar las matrículas de los vehículos que ingresan.

2. **Control del Tiempo:** El usuario podrá contabilizar el tiempo que llevan estacionados los vehículos para posteriormente generar un cobro preciso.  

3. **Dashboard:** Una vista que permite visualizar el resumen de cada día.  

4. **Historial:** La aplicación permitirá almacenar los datos de los últimos 365 días.
