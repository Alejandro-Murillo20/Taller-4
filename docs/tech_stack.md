# Stack Tecnológico

Este proyecto utiliza un ecosistema moderno centrado en el rendimiento y la escalabilidad en la nube.


## 🛠 Tecnologías Utilizadas


### Frontend

*   **React:** Biblioteca principal para la construcción de la interfaz de usuario basada en componentes.
*   **HTML5:** Estructura semántica de la aplicación.
*   **CSS3:** Estilizado avanzado y diseño responsivo.


### Backend & Infrastructure

*   **Firebase:** Plataforma integral para servicios de:
    *   *Authentication:* Gestión de usuarios.
    *   *Firestore / Realtime Database:* Persistencia de datos.
    *   *Hosting:* Despliegue y entrega de contenido.
